## Getting started

### 1. Install `rekit-studio`
```
npm install -g rekit rekit-studio
```

### 2. Start studio
```
rekit-studio -p 3333
```
then open http://localhost:3333 in your favorite browser.

### 3. Preview your work
Open in another console window:
```
# yarn install
yarn start
```
